# AXIOM v0.8.0 - Trust Kernel & AXIOM Trust Protocol

## Summary

AXIOM v0.8 introduces the Trust Kernel and the first public AXIOM Trust Protocol draft.

AXIOM can now answer:

- Where did this claim come from?
- What source introduced it?
- What trust policy scored it?
- What audit trail exists?
- Is it canonical, pending, flagged, rejected, or conflicted?
- Can another system validate the trust object?

## Highlights

### Trust Kernel

### Inspectability

## Correct product language

Use:

Every serious answer should come with a receipt.

Do not say:

- AXIOM proves absolute truth
- AXIOM eliminates hallucinations
- ATP is stable v1.0
- `.axiom` is runtime storage

Say:

- AXIOM exposes provenance, trust policy, audit trail, conflict status, and graph admission status.
- AXIOM reduces canonical graph pollution by quarantining flagged claims.
- `axiom-verify` validates ATP-compatible object structure and semantics.
